Règles du jeu
Le plateau est un grille 9×9 avec deux joueurs :

Rouge (Joueur 1, en haut)

Vert (Joueur 2, en bas)

Chaque joueur joue à tour de rôle :

Sélectionner un Pokémon de sa couleur

Déplacer vers une case vide (8 directions possibles)

Ou attaquer un Pokémon adverse adjacent

Combat :

Le plus rapide attaque en premier

Si l’adversaire survit, il contre-attaque

Les PV sont affichés en bas à gauche de chaque case

Victoire :

Si Mewtwo de l’adversaire est vaincu, la partie se termine


Quand un joueur gagne :

Un grand message " Joueur X a gagné !" apparaît

Deux boutons :

Rejouer → relance une nouvelle partie avec les positions de départ

Quitter → ferme le jeu
